# Poud_mining

ESX Extencion of mining script insipated by `Wasabi_Mining`

## Features

- Mining using ox_target
- Rock spawning as `prop_rock_2_f`
- Rocks disappear after mining and respawn after 7 seconds
- ox_lib skillcheck minigame before mining
- Pickaxe check
- Ore washing
- Ore smelting
- Material selling
- Server-side distance checks

## Installation

Ensure you have `ox_lib` and `es_extended` installed

Put script in your `resources` directory

Make sure all items from config are in items(Check `InventoryItems` folder for more examples)

Add ensure `Poud_mining` in your `server.cfg` after ox_lib & ox_inventory

## Dependencies
- `es_extended`
- `ox_lib`
- `ox_inventory`
